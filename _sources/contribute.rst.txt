Contributing to SPIRAL
======================

SPIRAL is an open source project and welcomes active participation in its development and maintenance efforts.  To work
directly with the source code, fork the repository of interest, work with your copy, then send a pull request to the SPIRAL
team.

Refer to the `Contributing to SPIRAL <https://github.com/spiral-software/spiral-software/blob/master/Contributing.md>`__ file for details.
