FFT with SIMD InstructionsÔÉÅ

This script generates an FFT that uses AVX intrinsics. Compare the first two lines to those in the previous simpler script.

opts := SIMDGlobals.getOpts(AVX_4x64f);
transform := TRC(DFT(16)).withTags(opts.tags);
ruletree := RandomRuleTree(transform, opts);
icode := CodeRuleTree(ruletree, opts);
PrintTo("AVX_DFT16.c", PrintCode("AVX_DFT16", icode, opts));

PrintTo() writes the code generated by PrintCode() into a file.
